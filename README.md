# Dissertation
Analyzing the impact of household environmental conditions on chronic diseases among India's elderly using LASI data with PCA-based index construction and logistic regression modeling.
Project Overview

This project examines how household environmental conditions influence chronic diseases among India's elderly population using data from the Longitudinal Ageing Study in India (LASI), Wave 1.

The study focuses on environmental factors such as cooking fuel, sanitation, drinking water, and housing quality and evaluates their association with chronic diseases including heart disease, cataract, and lung disease.

To measure environmental conditions, a Household Environment Quality Index (HEQI) was constructed using Principal Component Analysis (PCA).

Objectives

Analyze the relationship between household environmental conditions and chronic diseases.

Construct a Household Environment Quality Index (HEQI).

Identify key environmental risk factors affecting elderly health in India.

Dataset

Longitudinal Ageing Study in India (LASI), Wave 1

Sample size: 42,000+ households

Population: Individuals aged 45 years and above

Data includes household environment variables and health indicators.

Methodology
Data Processing

Data cleaning and variable recoding

Merging household and individual datasets

Index Construction

Principal Component Analysis (PCA) used to build the Household Environment Quality Index

Statistical Analysis

Descriptive statistics

Correlation analysis

Logistic regression models

Hypothesis testing

Tools & Technologies

STATA – Data management and statistical modeling

Excel – Descriptive analysis and visualization
